EF I Ensino Fundamental I 1º ao 5º ano
Quatro operações
Adição
a + b = soma
Some coluna por coluna da direita para a esquerda. Se uma coluna ultrapassar 9, "carregue 1" para a coluna seguinte.
Subtração
a − b = diferença
Se o dígito de cima for menor que o de baixo, peça emprestado à casa à esquerda (1 dezena = 10 unidades).
Multiplicação
a × b = produto
Propriedade comutativa: a × b = b × a. Distribua: a × (b + c) = a×b + a×c.
Divisão
a ÷ b = quociente
Prova real: quociente × divisor + resto = dividendo. Resto zero → divisão exata.
Prova Real da Multiplicação
Para verificar a × b = c: some os algarismos de c até obter um dígito, faça o mesmo com a e b. Se digito(a) × digito(b) ≡ digito(c) (mod 9), a conta está provavelmente correta.
Frações
Mesmos denominadores
a/n + b/n = (a+b)/n
Some ou subtraia apenas os numeradores. O denominador não muda.
Denominadores diferentes
a/b + c/d = (ad+bc)/bd
Multiplique cruzado os numeradores e coloque o produto dos denominadores em baixo. Simplifique ao final.
Multiplicação
(a/b) × (c/d) = ac/bd
Multiplique numerador com numerador, denominador com denominador. Simplifique antes se possível.
Divisão
(a/b) ÷ (c/d) = (a/b) × (d/c)
Inverta a fração divisora e multiplique. "Dividir por uma fração = multiplicar pelo inverso."
Simplificação
a/b → (a÷mdc) / (b÷mdc)
Divida numerador e denominador pelo MDC(a,b). A fração permanece equivalente.
Geometria básica
Perímetro do retângulo
P = 2(b + h)
b = base, h = altura. Soma de todos os lados.
Área do retângulo
A = b × h
Quadrado: A = l². Unidades de área: cm², m², km².
Perímetro do triângulo
P = a + b + c
Soma dos três lados. Triângulo equilátero: P = 3a.
Área do triângulo
A = (b × h) / 2
h é a altura perpendicular à base b.
Medidas e conversões
GrandezaConversões principais
Comprimento1 km = 1000 m = 100.000 cm = 1.000.000 mm
Massa1 t = 1000 kg = 1.000.000 g
Capacidade1 L = 10 dL = 100 cL = 1000 mL

Porcentagem
Calcular X% de N
resultado = N × X/100
Ex: 25% de 80 = 80 × 25/100 = 20.
Valores usuais
10% → ÷10
25% → ÷4
50% → ÷2
75% → ÷4×3
100% → o próprio
Aumento percentual
novo = N × (1 + X/100)
Ex: aumento de 15%: N × 1,15.
Desconto percentual
novo = N × (1 − X/100)
Ex: desconto de 20%: N × 0,80.
EF II Ensino Fundamental II 6º ao 9º ano
Equações do 1º grau
Forma geral
ax + b = 0 → x = −b/a
a ≠ 0. Isole o x: passe tudo que não é x para o outro lado, invertendo o sinal.
Inequação 1º grau
ax + b < 0 → x < −b/a
Ao multiplicar ou dividir por número negativo, inverta o sentido da desigualdade.
Como resolver: ax + b = cx + d
1. Agrupe os termos com x: ax − cx = d − b
2. Fatore: (a − c)x = d − b
3. Divida: x = (d − b) / (a − c), com a ≠ c
4. Prova real: substitua o valor encontrado na equação original
Regra de três
Direta (simples)
a/b = c/x → x = bc/a
Grandezas crescem juntas. Ex: mais km, mais litros de combustível.
Inversa (simples)
a·b = c·x → x = ab/c
Uma cresce enquanto a outra decresce. Ex: mais operários, menos dias.
Composta
x = b × (a₁/b₁) × (a₂/b₂) × …
Identifique para cada par se a relação é direta ou inversa. Diretas: divide. Inversas: inverte e divide.
Identificar o tipo
Pergunte: se a grandeza A aumenta, a grandeza B também aumenta? Se sim → direta. Se B diminui → inversa. Monte a proporção de acordo antes de calcular.
Razão e proporção
Razão
r = a/b (b ≠ 0)
Comparação entre duas quantidades de mesma unidade. Frequentemente expressa como fração irredutível.
Proporção
a/b = c/d ↔ ad = bc
Produto dos extremos = produto dos meios. Permite descobrir o quarto proporcional.
Quarto proporcional
x = (b × c) / a
Dada a proporção a:b = c:x, isolamos x multiplicando cruzado.
Potências e raízes
Propriedades
aᵐ · aⁿ = aᵐ⁺ⁿ
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
(aᵐ)ⁿ = aᵐⁿ
(ab)ⁿ = aⁿbⁿ
a⁰ = 1 (a ≠ 0)
a⁻ⁿ = 1/aⁿ
Raiz e potência fracionária
ⁿ√a = a^(1/n)
ⁿ√aᵐ = a^(m/n)
√a × √b = √(ab)
√a / √b = √(a/b)
√a é definida para a ≥ 0 nos reais.
Racionalização
1/√a = √a / a
Multiplique numerador e denominador pelo conjugado para eliminar raiz no denominador.

Teorema de Pitágoras e trigonometria no triângulo
Pitágoras
a² = b² + c²
a = hipotenusa (lado oposto ao ângulo reto). b, c = catetos.
Razões trigonométricas
sen θ = oposto / hipotenusa
cos θ = adjacente / hipotenusa
tg θ = oposto / adjacente
Ângulos notáveis
30°: sen=1/2, cos=√3/2, tg=√3/3
45°: sen=√2/2, cos=√2/2, tg=1
60°: sen=√3/2, cos=1/2, tg=√3
Relação fundamental
sen²θ + cos²θ = 1
Decorre diretamente do Teorema de Pitágoras aplicado ao círculo de raio 1.
Estatística básica
Média aritmética
x̄ = (x₁ + x₂ + … + xₙ) / n
Soma de todos os valores dividida pela quantidade de valores.
Moda
Mo = valor mais frequente
Pode ser amodal, bimodal ou multimodal.
Mediana
Md = valor central ordenado
Com n par: média dos dois valores centrais. Menos sensível a outliers que a média.
EM Ensino Médio 1ª a 3ª série
Equação do 2º grau — Bhaskara
Forma geral
ax² + bx + c = 0 (a ≠ 0)
Discriminante
Δ = b² − 4ac
Δ > 0 → 2 raízes reais. Δ = 0 → 1 raiz real (dupla). Δ < 0 → raízes complexas.
Fórmula de Bhaskara
x = (−b ± √Δ) / 2a
Soma e produto
x₁ + x₂ = −b/a
x₁ · x₂ = c/a
Relações de Girard. Úteis para montar equação a partir das raízes.
Roteiro de resolução
1. Identifique a, b, c
2. Calcule Δ = b² − 4ac
3. Analise o sinal de Δ
4. Se Δ ≥ 0: aplique x = (−b ± √Δ) / 2a
5. Verifique substituindo as raízes na equação original
Funções
Função afim (1º grau)
f(x) = ax + b
a = coeficiente angular (inclinação). b = coeficiente linear (interseção com eixo y). Raiz: x = −b/a.
Função quadrática (2º grau)
f(x) = ax² + bx + c
Parábola. Vértice V = (−b/2a, −Δ/4a). a > 0 → concavidade para cima; a < 0 → para baixo.
Função exponencial
f(x) = aˣ (a > 0, a ≠ 1)
Domínio: ℝ. Imagem: (0, +∞). a > 1 → crescente; 0 < a < 1 → decrescente.
Função logarítmica
f(x) = log_a(x)
Inversa da exponencial. Domínio: (0, +∞). Imagem: ℝ. Bases comuns: 10 (log) e e (ln).
Coeficiente angular
a = (y₂ − y₁) / (x₂ − x₁)
Taxa de variação da função afim. Equivale à inclinação da reta.
Progressões
PA — termo geral
aₙ = a₁ + (n−1)r
r = razão (diferença entre termos consecutivos). n = posição do termo.
PA — soma dos termos
Sₙ = n(a₁ + aₙ) / 2
Equivale a n × média aritmética entre o primeiro e o último termo.
PG — termo geral
aₙ = a₁ · qⁿ⁻¹
q = razão (quociente entre termos consecutivos). q ≠ 0.
PG — soma dos termos (q ≠ 1)
Sₙ = a₁(qⁿ − 1) / (q − 1)
Se q = 1: Sₙ = n·a₁.
PG infinita (|q| < 1)
S∞ = a₁ / (1 − q)
Converge apenas quando |q| < 1.
Trigonometria
Identidades fundamentais
sen²θ + cos²θ = 1
tg θ = sen θ / cos θ
sec²θ = 1 + tg²θ
csc²θ = 1 + cotg²θ
Adição de arcos
sen(A±B) = senA cosB ± cosA senB
cos(A±B) = cosA cosB ∓ senA senB
tg(A+B) = (tgA+tgB)/(1−tgA·tgB)
Arco duplo
sen 2A = 2 senA cosA
cos 2A = cos²A − sen²A
cos 2A = 1 − 2sen²A
tg 2A = 2tgA / (1−tg²A)
Lei dos senos
a/senA = b/senB = c/senC = 2R
R = raio da circunferência circunscrita.
Lei dos cossenos
a² = b² + c² − 2bc cosA
Generalização de Pitágoras. A = 90° → reduz-se ao Teorema de Pitágoras.
Logaritmos
Definição
log_a b = x ↔ aˣ = b
a > 0, a ≠ 1, b > 0.
Propriedades
log(xy) = log x + log y
log(x/y) = log x − log y
log xⁿ = n log x
log_a a = 1; log_a 1 = 0
Mudança de base
log_a b = log_c b / log_c a
Útil para calcular em calculadora (use log₁₀ ou ln).
Número e
e ≈ 2,71828…
ln(eˣ) = x
eˡⁿˣ = x
Base do logaritmo natural. Fundamental em crescimento e decaimento exponencial.
Análise combinatória
Princípio multiplicativo
total = n₁ × n₂ × … × nₖ
Se cada etapa é independente, multiplique o número de escolhas em cada etapa.
Fatorial
n! = n × (n−1) × … × 2 × 1
0! = 1
Permutação simples
P(n) = n!
Arranjos de todos os n elementos distintos.
Arranjo
A(n,p) = n! / (n−p)!
Escolha e ordena p elementos de n. A ordem importa.
Combinação
C(n,p) = n! / (p!(n−p)!)
Escolha p elementos de n sem importar a ordem. C(n,p) = C(n, n−p).
Binômio de Newton
(a+b)ⁿ = Σ C(n,k) aⁿ⁻ᵏ bᵏ
Termo geral: T(k+1) = C(n,k) aⁿ⁻ᵏ bᵏ.
Probabilidade
Probabilidade clássica
P(A) = |A| / |Ω|
|A| = casos favoráveis, |Ω| = total de casos igualmente prováveis.
Complementar
P(Aᶜ) = 1 − P(A)
União
P(A∪B) = P(A) + P(B) − P(A∩B)
Independentes
P(A∩B) = P(A) × P(B)
Apenas se A e B são eventos independentes.
Probabilidade condicional
P(A|B) = P(A∩B) / P(B)
Probabilidade de A dado que B ocorreu.

Geometria analítica
Distância entre pontos
d = √((x₂−x₁)² + (y₂−y₁)²)
Ponto médio
M = ((x₁+x₂)/2, (y₁+y₂)/2)
Equação da reta
geral: ax + by + c = 0
reduzida: y = ax + b
ponto-inclinação: y−y₀ = a(x−x₀)
Distância ponto-reta
d = |ax₀ + by₀ + c| / √(a²+b²)
Circunferência
(x−h)² + (y−k)² = r²
Centro (h, k), raio r.
Área do triângulo (coords.)
A = |x₁(y₂−y₃) + x₂(y₃−y₁) + x₃(y₁−y₂)| / 2
Matrizes e determinantes
Determinante 2×2
det = ad − bc
Onde a,b,c,d são os elementos da matriz 2×2.
Determinante 3×3 (Sarrus)
det = aei+bfg+cdh − ceg−bdi−afh
Repita as duas primeiras colunas à direita e some as diagonais.
Regra de Cramer (2×2)
x = Dₓ/D, y = D_y/D
D = det do sistema. Dₓ = det substituindo coluna de x pelos termos independentes.
Produto de matrizes
C[i,j] = Σₖ A[i,k] × B[k,j]
AB ≠ BA em geral. Linha de A × coluna de B.
ES Ensino Superior Cálculo, álgebra linear, EDO
Limites
Limites fundamentais
lim(x→0) sen x / x = 1
lim(x→0) (1−cos x)/x = 0
lim(x→∞) (1+1/x)ˣ = e
lim(x→0) (eˣ−1)/x = 1
lim(x→0) ln(1+x)/x = 1
Regra de L'Hôpital
lim f/g = lim f'/g'
Aplicável quando lim resulta em 0/0 ou ∞/∞. Derive numerador e denominador separadamente.
Continuidade
lim(x→c) f(x) = f(c)
Três condições: f(c) existe, limite existe, e ambos coincidem.
Ideia intuitiva
O símbolo dx pode ser lido como "um pedacinho infinitesimal de x". O limite é o processo de deixar esse pedacinho tender a zero e observar o comportamento da razão entre as variações — conforme descrito em Calculus Made Easy (Thompson, 1910).
Derivadas
Definição (taxa de variação)
f'(x) = lim(h→0) [f(x+h)−f(x)] / h
A derivada mede a taxa de variação instantânea de f em relação a x — a inclinação da reta tangente ao gráfico no ponto x.
Regras básicas
d/dx [c] = 0
d/dx [xⁿ] = nxⁿ⁻¹
d/dx [cf] = c·f'
d/dx [f+g] = f' + g'
Regra do produto
d/dx [fg] = f'g + fg'
Não basta multiplicar as derivadas separadamente.
Regra do quociente
d/dx [f/g] = (f'g − fg') / g²
Regra da cadeia
d/dx [f(g(x))] = f'(g(x)) · g'(x)
Se y = f(u) e u = g(x), então dy/dx = dy/du · du/dx.
Derivadas comuns
d/dx [eˣ] = eˣ
d/dx [ln x] = 1/x
d/dx [sen x] = cos x
d/dx [cos x] = −sen x
d/dx [tg x] = sec²x
d/dx [aˣ] = aˣ ln a
Significado geométrico
A derivada f'(x₀) é o coeficiente angular da reta tangente ao gráfico de f no ponto x₀. Se f'(x₀) = 0, o ponto pode ser máximo, mínimo ou ponto de inflexão — verifique com a segunda derivada.
Integrais
Integral indefinida
∫ f(x) dx = F(x) + C
F é a antiderivada de f: F'(x) = f(x). C é a constante de integração.
Teorema fundamental do Cálculo
∫ₐᵇ f(x) dx = F(b) − F(a)
Liga derivação e integração. A integral define a área sob a curva f entre a e b.
Integrais imediatas
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C (n≠−1)
∫ 1/x dx = ln|x| + C
∫ eˣ dx = eˣ + C
∫ sen x dx = −cos x + C
∫ cos x dx = sen x + C
∫ aˣ dx = aˣ/ln a + C
Integração por partes
∫ u dv = uv − ∫ v du
Mnemônico LIATE para escolha de u: Logaritmo, Inversa trig., Algébrica, Trig., Exponencial.
Substituição (u-sub)
∫ f(g(x)) g'(x) dx = ∫ f(u) du
Identifique u = g(x), calcule du = g'(x) dx e substitua.
Séries e expansões
Série de Taylor
f(x) = Σ f⁽ⁿ⁾(a)/n! · (x−a)ⁿ
Expansão de f em torno do ponto a. Série de Maclaurin: a = 0.
Expansões fundamentais (x=0)
eˣ = 1 + x + x²/2! + x³/3! + …
sen x = x − x³/3! + x⁵/5! − …
cos x = 1 − x²/2! + x⁴/4! − …
ln(1+x) = x − x²/2 + x³/3 − … (|x|≤1)
Série geométrica
Σ xⁿ = 1/(1−x), |x| < 1
Fórmula de Euler
eⁱᶿ = cos θ + i·sen θ
Caso especial θ=π: e^(iπ) + 1 = 0 (identidade de Euler).
Equações Diferenciais Ordinárias
Variáveis separáveis
dy/dx = f(x)g(y)
→ ∫ dy/g(y) = ∫ f(x) dx
Separe y e x em lados opostos e integre cada lado.
EDO linear 1ª ordem
y' + p(x)y = q(x)
μ(x) = e^(∫p dx)
Fator integrante μ: multiplique a EDO por μ, o lado esquerdo torna-se d(μy)/dx.
EDO 2ª ordem coef. constantes
ay'' + by' + cy = 0
equação característica: ar² + br + c = 0
Δ > 0: y = C₁e^(r₁x) + C₂e^(r₂x). Δ = 0: y = (C₁+C₂x)e^(rx). Δ < 0: solução oscilatória com senos e cossenos.
Lei do crescimento/decaimento
dy/dt = ky → y = y₀ eᵏᵗ
k > 0 → crescimento. k < 0 → decaimento. Meia-vida: t₁/₂ = ln2/|k|.
Álgebra linear
Produto escalar
u·v = |u||v| cos θ = Σ uᵢvᵢ
u ⊥ v ↔ u·v = 0.
Produto vetorial (ℝ³)
|u×v| = |u||v| sen θ
Direção: regra da mão direita. |u×v| = área do paralelogramo formado por u e v.
Autovalores
det(A − λI) = 0
Resolva o polinômio característico para obter λ. Substitua para encontrar os autovetores.
Espaço nulo e posto
posto(A) + nulidade(A) = n
n = número de colunas. Teorema do núcleo e da imagem.
Gram-Schmidt
eₖ = vₖ − Σᵢ (vₖ·eᵢ/|eᵢ|²) eᵢ
Ortogonaliza um conjunto de vetores linearmente independentes.
Números complexos
Forma algébrica
z = a + bi, i² = −1
a = parte real, b = parte imaginária.
Módulo e argumento
|z| = √(a²+b²)
θ = arg(z) = arctan(b/a)
Forma polar
z = r(cos θ + i sen θ) = reⁱᶿ
Multiplicação polar
z₁z₂ = r₁r₂ · eⁱ⁽ᶿ¹⁺ᶿ²⁾
Módulos multiplicam, ângulos somam.
Fórmula de De Moivre
zⁿ = rⁿ(cos nθ + i sen nθ)
Conjugado
z̄ = a − bi
z · z̄ = |z|²
Útil para dividir complexos: multiplique numerador e denominador pelo conjugado do denominador.
